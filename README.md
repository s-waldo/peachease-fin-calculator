# Baby Steps Calculator

Baby Steps Calculator is a tool for financial coaches to visually engage their clients into the baby steps.

## Description

This tool is to be utilized by financial coaches in their discovery sessions with prospects.  This program will ask a series of questions and project out, if the client sticks to the baby steps, what a projected outcome will be in the future.  Financial Coaches often have the problem of not being able to show their clients an outcome for THEIR future.  This program aims to solve that for financial coaches to help them gain more clients.

## Tech Stack

* MVP: React
* Frontend: React (ReactRouter)
* Backend: Laravel
* Database: MySQL
